Contributing

Tracking Changes
----------------

We use a changelog file to track changes and updates to our project. You can find our `CHANGELOG.md <https://github.com/darosio/clophfit/blob/main/CHANGELOG.md>`_ file here.

Our changelog follows the `Keep a Changelog <https://keepachangelog.com/en/1.0.0/>`_ format, and we tag releases according to the `Semantic Versioning <https://semver.org/>`_ scheme.
